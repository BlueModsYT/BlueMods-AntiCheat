## **ü§ñ COMMANDS FOR ANTICHEAT**
‚ñ† !help - shows the list of the commands.<br>
‚ñ† !about - shows the developer of the addon.<br>

Spawn Commands<br>
‚ñ† !spawn - go back to lobby.<br>
‚ñ† !rspawn - remove the current spawn set.<br>
‚ñ† !setspawn - make a setspawn to able to use !spawn.<br>

TPA Commands<br>
‚ñ† !tpa send \<player> - allows you to request a teleport from any players.<br>
‚ñ† !tpa accept - allows you to accept a current request from other player.<br>
‚ñ† !tpa decline - you can only decline a request from other player.<br>
‚ñ† !tpa cancel - you can cancel a request from the player.<br>
‚ñ† !tpa block \<player> - allows you to block a player from sending a teleport request.<br>
‚ñ† !tpa unblock \<player> - unblock the blocked player so they can able to send a request to you again.<br>

Gamemode Commands<br>
‚ñ† !gma \<player> - change gamemode to adventure.<br>
‚ñ† !gmc \<player> - change gamemode to creative.<br>
‚ñ† !gms \<player> - change gamemode to survival.<br>
‚ñ† !gmsp \<player> - change gamemode to spectator.<br>
‚ñ† !vanish \<player> - makes yourself invisibility.<br>

Moderation Commands<br>
‚ñ† !kick \<player> \<reason> - kick a specific player in server.<br>
‚ñ† !ban add \<player> \<reason> - ban a specific player in server.<br>
‚ñ† !ban remove \<player> - unban a specific player in server.<br>
‚ñ† !ban list - see the list of the banned players on the server.<br>
‚ñ† !cmdsf <enable/disable> - disabled command block logs and popping on chats. [MODULE]<br>
‚ñ† !mute <add/remove> \<player> - mute a specific player in server.<br>
‚ñ† !mute list - see the list of muted user.<br>
‚ñ† !freeze <add/remove> \<player> - freeze a specific player.<br>
‚ñ† !freeze list - see the list of freezed user.<br>
‚ñ† !lagclear <default/mobs/all> - kill all items on the ground.<br>
‚ñ† !ecwipe \<player> - allows you to remove items on their ender_chest.<br>
‚ñ† !give \<item> \<value> [data] - give yourself an item(s).<br>
‚ñ† !troll \<troll> \<player> - troll someone in the server.<br>
‚ñ† !welcome <add/remove> [text] - add or remove specific set text.<br>
‚ñ† !banitem <add/remove> <item> - add or remove ban items.<br>
‚ñ† !banitem list - see the list of banned.<br>
‚ñ† !clearchat - clear your chat (only you can see it)<br>

Operator Commands<br>
‚ñ† !op <add/remove> \<player> - op a specific player in server to immune to any anticheat.<br>
‚ñ† !op list - see the list of the admins on the anticheat.<br>
‚ñ† !notify <add/remove> \<player> - notification when someone got flagged by the anticheat.<br>
‚ñ† !notify list - see the list of the notify on the anticheat.<br>
‚ñ† !trusted <add/remove> \<player> - add trusted or remove on specific player.<br>
‚ñ† !trusted list - see the list of the trusted on the anticheat.<br>
